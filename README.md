# Developmental Compass — Static Site

This is a simple, single-file static website styled with Tailwind (via CDN) and using two images in `assets/`.
It’s ready for GitHub Pages.

## Deploy on GitHub Pages (user/organization site)
1. Create (or use) a repo named `<your-username>.github.io`.
2. Upload `index.html` and the `assets/` folder to the repo root.
3. Visit `https://<your-username>.github.io` after a minute or two.

## Deploy on GitHub Pages (project site for a custom domain)
1. Create any repo name (e.g., `devcomp-site`), push these files to the `main` branch.
2. In **Settings → Pages**, set **Source** to **Deploy from a branch**, and select `main` / `/root`.
3. (Optional) Add a `CNAME` file containing your custom domain (e.g., `developmentalcompass.com`).
